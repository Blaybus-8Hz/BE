# ✨ 8Hz 프로젝트
8Hz는 실패없는 헤어스타일 변화를 위한 퍼스널 헤어스타일 컨설팅 예약 모바일 웹앱 서비스로,<br/>
헤어 디자이너와 고객 간의 예약 및 결제를 간편하게 처리할 수 있도록 설계되었습니다.

>📌 배포링크 : https://www.haertz.kr/
